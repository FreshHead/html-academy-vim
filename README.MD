# html-academy-vim
Расширение для хромоподобных браузеров, включающие VIM-mode для тренажёров html-academy.ru

## Клавиатурные сочетания

- **Shift+H** - открыть теорию (`.course-theory`)
- **Shift+Enter** - кнопка "Далее" (`.course-goals__button--next`)
- **Shift+J** - фокус на HTML редактор (`#html-editor`)
- **Shift+K** - фокус на CSS редактор (`#css-editor`)
