# html-academy-vim
Расширение для хромоподобных браузеров, которое включает VIM-mode для тренажёров html-academy.ru

## Клавиатурные сочетания

- **Shift+H** - открыть теорию (`.course-theory`)
- **Shift+Space** - кнопка "Далее" (`.course-goals__button--next`)
- **Shift+J** - фокус на HTML редактор (`#html-editor`)
- **Shift+K** - фокус на CSS редактор (`#css-editor`)
- **Shift+L** - фокус на JS редактор (`#js-editor`)
